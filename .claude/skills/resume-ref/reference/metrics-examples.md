| 패턴 | 예시 |
|------|------|
| 일/월 단위 | "일 PV 2,000만", "MAU 1,500만" |
| 동시 접속 | "동시 접속자 10만 처리" |
| 데이터 규모 | "일 100GB 데이터 처리" |
